EPIC can be used with Azure AKS, it just requires the installation of the Gateway Controller as per the getting [Started Guide]({{< ref "/gateway_service/user_manual/getting-started" >}}).


### Tested Versions:

Kubernetes: 1.22.6 & 1.23.3

CNI: Kubenet & Azure CNI

### Notes:

No specific configuration or implementation was required for support of Azure AKS
